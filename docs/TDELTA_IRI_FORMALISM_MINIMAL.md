# TΔ / IRI / Ω(t) — Minimal Operational Formalism

## Purpose

Define when two initially close descriptions stop being structurally equivalent.

No semantics.
No prediction.
No interpretation.

Only structure.

---

## 1. Divergence

Given two trajectories:

x1(t), x2(t)

Δ(t) = d(x1(t), x2(t))

d must be explicit.

---

## 2. Structural Divergence Time (TΔ)

TΔ = min t such that:

Δ(t:t+w) is no longer structurally stable

Interpretation:

First time where equivalence breaks.

---

## 3. Irreversibility Index (IRI)

IRI = (1/T) Σ 1[Δ(t) > τ]

τ = recovery threshold

Interpretation:

Persistence of divergence.

---

## 4. Local Structural Coherence Ω(t)

Ω(t) = αC(t) + βK(t) + γ(1 − N(t))

C = compressibility  
K = coherence  
N = noise sensitivity  

α + β + γ = 1

---

## 5. Hierarchy

Ω(t) → TΔ → IRI → LIMIT

---

## 6. Core Principle

High coherence does NOT imply identity.

invariance ≠ identity

---

## 7. Function

OMNIA does not validate truth.

It detects when equivalence is no longer justified.
