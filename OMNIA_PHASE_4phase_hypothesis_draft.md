## Purpose

This draft defines the reduced four-phase version of OMNIA-PHASE.

It exists because the previous reduction test showed that:

```text
SATURATION is the weakest phase in the current five-phase system

So this draft asks a narrower question:

Does OMNIA-PHASE become clearer if SATURATION is removed as a standalone phase?

This is not yet a final decision. It is a competing hypothesis.


---

Reduced phase set

The reduced system contains:

1. BIFURCATION


2. CONSOLIDATION


3. OPPORTUNITY


4. STOP



In this reduced model:

SATURATION is not treated as a full standalone phase

Instead, it is treated as:

a transition band toward STOP


---

Why reduce the system

The current five-phase model is expressive, but it has one weak point:

SATURATION risks overlapping too strongly with

collapsing opportunity

rising stop pressure

diminishing recoverability



That means it may not justify full phase status.

A smaller system may therefore be:

cleaner

less redundant

easier to interpret

more stable conceptually



---

Four-phase interpretation

1. BIFURCATION

The process is structurally splitting.

Dominant profile:

instability high

coherence not yet dominant

local organization contested


Main ecosystem relation:

OMNIAMIND



---

2. CONSOLIDATION

The process is becoming locally coherent and structurally persistent.

Dominant profile:

coherence high

instability reduced

structure more stable under transformation


Main ecosystem relation:

OMNIA



---

3. OPPORTUNITY

The process still contains useful residual structural space.

Dominant profile:

opportunity high

recoverability high

continuation still justified


Main ecosystem relation:

OMNIA-RADAR



---

4. STOP

The process has reached a state where continuation is no longer justified.

Dominant profile:

stop pressure dominant

opportunity near zero

recoverability near zero or irrelevant


Main ecosystem relation:

OMNIA-LIMIT



---

What happens to SATURATION

In the four-phase hypothesis, SATURATION is not deleted as a phenomenon.

It is reinterpreted.

Correct reading:

SATURATION is a transition condition, not necessarily a full phase

That means:

opportunity may weaken

recoverability may fall

stop pressure may rise


But this may not require a separate phase name.

Instead, saturation may simply describe the final deterioration of OPPORTUNITY on the way to STOP.


---

Reduced transition path

The dominant reduced path becomes:

BIFURCATION -> CONSOLIDATION -> OPPORTUNITY -> STOP

This is still not strictly linear. But it is cleaner than the five-phase version.

Possible non-linear transitions remain:

BIFURCATION -> OPPORTUNITY

BIFURCATION -> STOP

CONSOLIDATION -> STOP

OPPORTUNITY -> CONSOLIDATION

STOP -> BIFURCATION in rare reopening cases


So the system remains graph-like, not rigidly sequential.


---

Comparison with the five-phase model

Aspect	Five-phase model	Four-phase model

Expressive richness	Higher	Slightly lower
Redundancy risk	Higher	Lower
Clarity	Moderate	Higher
Boundary around STOP	More nuanced	More direct
Conceptual stability	Weaker due to SATURATION	Stronger if SATURATION is non-essential



---

Main gain of the four-phase model

The gain is not simplicity for its own sake.

The gain is:

removing the weakest phase if it does not justify its own existence

This improves discipline.

A phase system should not keep a phase alive only because it sounds plausible.

It should keep it only if it adds real explanatory value.


---

Main risk of the four-phase model

The risk is loss of boundary nuance.

If SATURATION is removed too early, the system may lose the distinction between:

declining but still meaningful continuation

continuation already structurally unjustified


So the four-phase model is cleaner, but may also be harsher.

This is its main weakness.


---

Current reading

At this stage, the four-phase hypothesis appears stronger than the five-phase model in terms of:

compactness

readability

non-redundancy


But weaker in terms of:

transitional nuance near STOP


So the real question becomes:

Is transitional nuance worth an entire extra phase?

That is now the central issue.


---

Provisional verdict

Current provisional verdict:

The four-phase model is the stronger default hypothesis
unless SATURATION later proves necessary as a distinct boundary phase

This is the cleanest current reading.


---

Practical consequence

If OMNIA-PHASE is ever developed further, it should proceed under this rule:

Default hypothesis

4 phases

Burden of proof

SATURATION must justify itself to re-enter as a full standalone phase

This is the correct discipline.


---

What this draft achieves

This draft closes the current reduction problem more cleanly.

Instead of only saying:

SATURATION is suspicious

it now provides a serious competing structure:

BIFURCATION
CONSOLIDATION
OPPORTUNITY
STOP

That makes the phase problem sharper and more falsifiable.


---

Minimal conclusion

The four-phase hypothesis is currently the strongest reduced version of OMNIA-PHASE.

It preserves:

instability

coherence

residual useful space

terminal stop


while treating SATURATION as a transition band rather than a full phase.

Current status:

cleaner
stronger
still private
not yet canonical
