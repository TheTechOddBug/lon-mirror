## Purpose

This draft compresses the current OMNIA-PHASE hypothesis into a single readable matrix.

It does not define equations.
It does not define thresholds.
It does not define a classifier.

Its role is narrower:

- reduce the current phase system to one compact visual structure
- align phase names with dominant signal profiles
- make the draft easier to test for internal coherence

This remains outside all repositories.

---

## Core principle

A phase is not a label alone.

A phase is a **signal configuration**.

That means:

- each phase corresponds to a recognizable structural profile
- the same signal may appear in multiple phases, but with different dominance
- the phase identity comes from the whole pattern, not from one variable in isolation

This matrix exists to make that pattern visible.

---

## Minimal signal set

The current draft uses six structural signals:

1. **Instability**
2. **Coherence**
3. **Opportunity**
4. **Recoverability**
5. **Saturation Pressure**
6. **Stop Pressure**

These remain qualitative in v0.

---

## Readout rules

The matrix should be read through dominance, not literal absolutism.

### High
The signal is structurally central to the phase.

### Mid
The signal is present but not dominant.

### Variable
The signal may be high or low depending on local path shape.
It is not phase-defining by itself.

### Falling / Rising
The signal is better understood as a directional trend than as a fixed level.

### Near zero
The signal has effectively collapsed for practical interpretation.

### Irrelevant / terminal
The signal no longer helps distinguish the state meaningfully because the process is already structurally ended or terminally defined.

## Matrix interpretation by phase

### BIFURCATION

Key pattern:

- instability dominates
- coherence is not yet strong
- opportunity and recoverability remain unresolved
- stop is not yet dominant

This is the phase of active splitting.

---

### CONSOLIDATION

Key pattern:

- coherence dominates
- instability falls
- structure persists
- stop pressure remains weak

This is the phase of local structural stabilization.

---

### OPPORTUNITY

Key pattern:

- opportunity dominates
- recoverability remains high
- instability is not central
- saturation has not yet become structurally strong

This is the phase where continuation is still worth the cost.

---

### SATURATION

Key pattern:

- opportunity weakens
- recoverability falls
- saturation pressure rises
- stop pressure begins to emerge

This is the phase where continuation still exists but becomes progressively less justified.

---

### STOP

Key pattern:

- stop pressure dominates
- opportunity is effectively gone
- recoverability is absent or irrelevant
- other signals no longer matter enough to justify continuation

This is the terminal structural phase.

## Matrix as a process reading

The matrix also suggests the following dominant movement:

```text id="503597"
BIFURCATION
-> CONSOLIDATION
-> OPPORTUNITY
-> SATURATION
-> STOP

But this should not be interpreted as rigid chronology.

It is only the simplest dominant path.

The matrix still allows:

unstable but recoverable opportunity

direct collapse from bifurcation to stop

stable but exhausted consolidation

rare reopening from stop or saturation


So the matrix is process-readable, but not linearistic.

Why this matrix matters

This matrix is useful because it makes three things visible at once:

1. the phases


2. the dominant signals


3. the current module alignment



Without it, OMNIA-PHASE remains too verbal.

With it, the draft becomes more inspectable.


---

Immediate tests for coherence

The matrix is worth continuing only if it passes at least these internal tests:

1. Distinctiveness

Each phase must have a visibly different dominant profile.

2. Non-redundancy

The five phases must not collapse trivially into two or three hidden super-states.

3. Module compatibility

The current module ecosystem must map into the matrix without forced distortion.

4. Explanatory gain

The matrix must explain something the modular reading alone does not explain as clearly.

If these fail, the draft should not be promoted.


---

Immediate weaknesses

This matrix is still weak because:

the signals are qualitative

the boundaries between phases remain fuzzy

no transition scoring exists yet

no synthetic phase runner exists yet

no empirical trace mapping exists yet


So this is still a scaffold, not a final framework.


---

Promotion rule

This draft should remain private until at least one of the following exists:

a small synthetic phase classifier

a transition scoring model

a reduced signal set that survives simplification

clear evidence that the matrix improves ecosystem readability


Without that, this should stay outside all repos.
